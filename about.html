    About

    A brief summary on what the checker looks for.

    This does a basic check on the accessibility of the PDFs uploaded. This includes :
    • Whether or not the document is marked as a "Tagged PDF"
    • Counts the different type of heading tags marked in the document
    • Marks whether or not any content tagged as a 'Figure' has alternative-text
    • Whether or not the document has a main document language specified

    Structure Tree

    It is within this structure that the logical structure or the hierarchy of tagged content of the document is held. Based of its existence, one can tell whether or not the document is tagged. This structure is traversed in order to find tags for headings and figures.

    Document Properties

    There are basic properties of documents that should be set in the document's metadata.
    • Title
    • Author
    • Keywords
    • Subject
    XMP for PDF Documents has been introduced in Acrobat 5. It is a a powerful format for carrying metadata, that will survive between format conversions much better than the document information entries, which can be edited/viewed in this way.
